# The Build Up Show

A lead generation and outreach system for interviewing lesser-known, profitable founders.

We feature founders making real money with real businesses — not unicorn fairy tales.

**Email:** hello@thebuildupshow.com

## What's Inside

```
leads/          - Founder lead lists by category
outreach/       - Outreach playbook and strategy
templates/      - Email, LinkedIn, and Twitter/X message templates
```

## Who We Target

- Micro-SaaS founders doing $1K-$100K+ MRR
- Chrome extension and Shopify app builders
- Small e-commerce brands and physical product businesses
- Service businesses that scaled from nothing
- Food/bev brands in local and national retail
- Content creators and newsletter operators

## Weekly Workflow

1. **Monday** - Research 10 new leads, add to leads folder
2. **Tuesday-Thursday** - Send outreach (email + LinkedIn + Twitter)
3. **Friday** - Follow up on pending, update statuses
4. **Weekend** - Engage with founder content on social media
